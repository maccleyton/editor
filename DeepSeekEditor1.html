Como usar o editor:

  • Fórmulas LaTeX inline: Use $...$ para fórmulas inline, como $E = mc^2$
  • Fórmulas LaTeX em bloco: Use $$...$$ para fórmulas em bloco
  • Código inline: Use `...` para código inline, como `console.log()`
  • Blocos de código: Use três crases (```) para criar blocos de código
  • Blocos flutuantes: Use o botão "Inserir Bloco Flutuante" para adicionar conteúdo destacado

Exemplo de uso:

Para resolver a equação quadrática, usamos:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

Em JavaScript, podemos implementar isso como:

function quadratic(a, b, c) {
    const discriminant = b * b - 4 * a * c;
    if (discriminant < 0) return null;
    
    const sqrtDiscriminant = Math.sqrt(discriminant);
    return [
        (-b + sqrtDiscriminant) / (2 * a),
        (-b - sqrtDiscriminant) / (2 * a)
    ];
}
